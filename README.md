# DAWnet CLI
A python CLI used to setup &amp; manage [DAWnet](https://dawnet.tools/) remote scripts on MAC, LINUX and Servers

### Requirements:

- ensure Python >= 3.x is installed

```python
python --version
```

- ensure Pip is installed

```python
pip list
```

### Installation:
 
```python
sudo pip install -U dawnet-cli
```

### Usage:

```python
dawnet -h
```
